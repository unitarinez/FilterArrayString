# FilterArrayString

Как использовать:

Запустите программу.
Введите строки через пробел.
Программа выведет новый массив строк, удовлетворяющий условию (длина <= 3).
Пример использования:

Введите строки через пробел:
apple banana cat dog elephant
Вывод:

Новый массив строк (длина <= 3):
cat
dog

Замечания:

Программа не использует коллекции, только массивы.
В случае отсутствия строк, удовлетворяющих условию, выводится пустой массив.
